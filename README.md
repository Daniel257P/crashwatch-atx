# CrashWatch ATX

This repository is for my final personal project CrashWatch ATX. CrashWatch ATX is a web application that centralizes and visualizes the most 
common types of traffic accidents in Austin, TX through interactive heat maps and street
level imagery, allowing users to analyze daily, weekly, monthly, and annual patterns to 
support road safety decisions and urban planning. 
This idea comes from my experience contributing to Hivemapper and from the high road 
fatality rate in the Dominican Republic.

## APIs Used
- [City of Austin Open Data — Traffic Incident Reports](https://data.austintexas.gov/resource/dx9v-zd7x.json)
- [Austin Crash Report Data — Crash Level Records](https://data.austintexas.gov/resource/y2wy-tgr5.json)
- [Beemaps API](https://beemaps.com) — Street-level imagery
- [GNews API](https://gnews.io) — Related news by location
